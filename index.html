KDScript

KDScript is a custom script module for the Dark Engine games Thief: The Dark Project and Thief II: The Metal Age.

  • This is the documentation for version 0.7.0, released on 2013-09-01.
  • You can download the latest version of the module from its thread on TTLG.
  • If you want step-by-step instructions for using the scripts, go to the quick start section.
